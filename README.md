Delhi Metro Analysis Dashboard

Project Overview

This project focuses on analyzing the Delhi Metro Network using data analytics and visualization techniques.
It covers the complete workflow from data cleaning → analysis → visualization → dashboard creation.

The goal is to extract meaningful insights such as:

Station distribution across metro lines
Infrastructure types (Elevated, Underground, At-Grade)
Growth of metro stations over time
Distance patterns between stations
