Delhi Metro Analysis Dashboard
Project Overview

This project analyzes the Delhi Metro network using data analytics and visualization techniques. It covers the complete workflow from data cleaning to analysis, visualization, and dashboard creation.

The objective is to extract meaningful insights, including:

Station distribution across metro lines
Infrastructure types (Elevated, Underground, At-Grade)
Growth of metro stations over time
Distance patterns between stations
